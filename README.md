# Personal Portfolio - Andrew Ceaser Mwaura Mungai

A professional portfolio website for Andrew Ceaser Mwaura Mungai, a Civil & Construction Engineer with 17+ years of experience delivering high-value infrastructure projects across East Africa.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Sections**: Hero, About, Experience Timeline, Key Projects, Skills & Certifications, Contact
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Scroll Animations**: Elements animate into view as you scroll
- **Animated Counters**: Key statistics animate on page load
- **Back to Top Button**: Quick navigation back to the top of the page

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Inter, Playfair Display)
- Font Awesome Icons

## Structure

```
├── index.html      # Main HTML file
├── styles.css      # Stylesheet
├── script.js       # JavaScript functionality
└── README.md       # Documentation
```

## Getting Started

Simply open `index.html` in a web browser or deploy to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).

## Deployment

This site can be deployed to GitHub Pages:
1. Go to repository Settings > Pages
2. Set source to "Deploy from a branch"
3. Select `main` branch and `/ (root)` folder
4. Click Save
